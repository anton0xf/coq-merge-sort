## HowTo
Regenerate Makefile:
```sh
$ coq_makefile -f _CoqProject -o Makefile
```
or
```sh
$ make Makefile.conf
```

clean project:
```sh
$ make clean
# or even
$ make cleanall
```

build coq file:
```sh
$ make ${FILE_NAME}.vo
```

build all:
```sh
$ make
```

build and run OCaml code with generated by `GenOcaml.v` implementation of `merge_sort` function:
```sh
$ make
$ cd ocaml
$ make run
```
clean OCaml code:
```sh
# in /ocaml dir
$ make clean
```
